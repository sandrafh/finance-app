## Get started

### Requirements
- nvm
- npm
- node 

1. Enter to the project folder

   ```bash
   cd finance-app
   ```
   
2. Use Node version

   ```bash
   nvm use
   ```

3. Install dependencies

   ```bash
   npm install
   ```

4. Install XCode or Android Studio

5. Install a simulator if you want to use it

6. Make a dev build for Android or iOS
    ```bash
   npm run ios
   ```
    Android will need java runtime environment, version >=17  is required.
    A java management tool like sdkman is recommended to install java. (https://sdkman.io/)
    ```bash
   npm run android
   ```

7. Start the local server
  ```bash
   npm start
   ```

Follow the terminal instructions to run on iPhone, Android or Web
